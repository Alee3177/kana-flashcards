促音規則

っ = 停半拍

看到小小的 っ，中間停一下，例如：きて → きって。

常用促音單字

長音規則

長音 = 拉長一拍

例如：おばさん → おばあさん。多一拍，意思可能完全不同。

型式 例子
あ → ああ おかあさん
え → えい せんせい
お → おう とうきょう

常用長音單字

濁音・半濁音規則

濁點「゛」與半濁點「゜」

清音加 ゛ 變濁音，例如 か → が；は行加 ゜ 變半濁音，例如 は → ぱ。

常用濁音・半濁音單字

拗音規則

い段 + 小ゃ・小ゅ・小ょ

拗音是「一個大音 + 一個小音」組成，例如 き + ゃ → きゃ / kya。小字要寫小一點，發音合成一拍。
